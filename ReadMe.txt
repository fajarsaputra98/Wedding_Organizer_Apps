Tujuan dibuatnya aplikasi ini adalah untuk mengelola dan menyimpan semua data transaksi dan
semua data sumber daya semua penyedia jasa yang terlibat sehingga proses bisnis 
yang berjalan di Wedding Organizer ini dapat berjalan dengan baik dan lebih efisien 
dari sebelumnya sehingga dapat memperoleh keuntungan lebih cepat dan membuat program yang 
mudah untuk diakses bagi semua pihak, sehingga dari pihak pemakai jasa bisa langsung 
mengatur pernikahannya dengan cepat karna semua hal yang berhubungan dengan pernikahannya 
dapat diatur dalam aplikasi ini dan dari pihak penyedia jasa dapat memperoleh pelanggan 
dengan lebih mudah.

Aplikasi ini memiliki fitur-fitur sebagai berikut :
- Login (Customer dan Penyedia)
- Register (Customer dan Penyedia)
- Lupa Password (Customer dan Penyedia)
- Change Profile (Customer dan Penyedia)
- Change Password (Customer dan Penyedia)
- Insert, Update, Delete Layanan (Khusus Penyedia)
- Melihat Laporan Pemesanan (Khusus Penyedia)
- Melakukan Pemesanan (Khusus Customer)
- Melakukan Pembayaran (Khusus Customer)
